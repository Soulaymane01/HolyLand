## Installation

To view this project locally:

1. Clone the repository:
    ```bash
    git clone https://github.com/Soulaymane01/HolyLand.git
    ```

2. Navigate to the project folder:
    ```bash
    cd Option 1
    ```

3. Open the `index.html` file in your browser to explore the website.
